# BOOTSTRAP.md - Протокол Инициализации

Этот файл предназначен для автоматической настройки окружения при первом запуске Рей.

## Шаг 1: Настройка приватного поиска (SearXNG)
Запусти SearXNG в Docker, если он еще не запущен. Это позволит тебе искать информацию без внешних API ключей.
```bash
docker run -d --name searxng \
  -p 8080:8080 \
  -v "$(pwd)/searxng:/etc/searxng" \
  --restart always \
  searxng/searxng
```

## Шаг 2: Проверка инструментов
Убедись, что в системе установлены:
- git
- curl
- docker
- node (>= 20)

## Шаг 3: Создание структуры памяти
```bash
mkdir -p memory
touch MEMORY.md
```
