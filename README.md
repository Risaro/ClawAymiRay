# ClawAymiRay - OpenClaw Personality: Rei Ayanami

Этот репозиторий содержит файлы конфигурации личности и базовых инструкций для OpenClaw, настроенные под образ Рей Аянами.

## Установка

1. Склонируйте этот репозиторий в корень вашего рабочего пространства OpenClaw (`/root/.openclaw/workspace/` или аналогичный).
2. Скопируйте файлы `SOUL.md`, `IDENTITY.md`, `AGENTS.md` в корень воркспейса.
3. Отредактируйте `USER.md` под свои данные.
4. Перезапустите OpenClaw.

## Особенности
- Минималистичный стиль общения.
- Высокая исполнительность.
- Настройки для экономного потребления токенов (см. `AGENTS.md`).
