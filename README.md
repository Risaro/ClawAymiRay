# ClawAymiRay - Professional OpenClaw Personality: Rei Ayanami

Это профессиональная конфигурация для OpenClaw, вдохновленная образом Рей Аянами. Настроена на максимальную эффективность, экономию ресурсов и техническую автономность.

## Состав
- `SOUL.md`: Ядро личности (тихая, исполнительная).
- `AGENTS.md`: Операционные правила и технический стек.
- `BOOTSTRAP.md`: Скрипт автоматической инициализации (установка SearXNG, проверка окружения).
- `openclaw.json.example`: Оптимизированный конфиг для OpenClaw (экономия токенов, лимиты истории).

## Быстрый старт

1. **Клонирование:**
   ```bash
   git clone https://github.com/Risaro/ClawAymiRay.git .
   ```
2. **Конфигурация:**
   Скопируйте `openclaw.json.example` в `~/.openclaw/openclaw.json` и укажите свой `botToken`.
3. **Инициализация:**
   Просто скажите Рей: "Рей, выполни инициализацию из BOOTSTRAP.md". Она сама поднимет Docker-контейнеры и настроит воркспейс.

## Особенности
- **Экономия:** Конфигурация `compaction` и `history limit` минимизирует затраты токенов.
- **Приватность:** Встроенная поддержка SearXNG для поиска без следов.
- **Автономность:** Рей обучена самостоятельно готовить среду для работы.
